Allow list: A security strategy organisations can use that approves a list of email addresses, IP addresses, domain names or applications, while denying all others, to protect computers and networks from threats.

Anti-virus/Malware: Software that is designed to detect, stop and remove viruses and other kinds of malicious software.

Attack Vector: The method an attacker uses to gain unauthorised access to a computer system or device. So brute force attacks or virus/Malware, etc.

Brute force attacks: An attack that uses computing power to automatically enter a large number of guesses for security credentials in an attempt to gain unauthorised access. so hacker sitting down( to be fair its mostly done by ai and bots now) and trying out to guess the login information to log in by inputing random login and password thousands of time

Cyber Attack:An attempt to damage, disrupt or gain unauthorised access to computer systems, networks or devices.
